# Задание

---

Вашей программе на вход подаются две строки **s** и **t**, состоящие из строчных латинских букв.

Выведите одно число – количество вхождений строки **t** в строку **s**.

**Пример:</br>
s = "abababa"</br>
t = "aba"**</br>

Вхождения строки **t** в строку **s**:</br>
***aba***baba</br>
ab***aba***ba</br>
abab***aba***</br>

---

**Sample Input 1:**</br>
abababa</br>
aba</br>

---

**Sample Output 1:**</br>
3</br></br></br>

---

**Sample Input 2:**</br>
abababa</br>
abc</br>

---

**Sample Output 2:**</br>
0</br></br></br>

---

**Sample Input 3:**</br>
abc</br>
abc</br>

---

**Sample Output 3:**</br>
1</br></br></br>

---

**Sample Input 4:**</br>
aaaaa</br>
a</br>

---

**Sample Output 4:**</br>
5</br>
